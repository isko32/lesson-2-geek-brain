## Списки

чтобы добавить списки без нумерации ,необходимо пункты выделить (*). Пример:
* Элемент 1
* Элемент 2

чтобы добавить списки с нумерацией, необходимо просто пронумеровать пункты. Например:
1. Элемент 1
2. Элемент 2
