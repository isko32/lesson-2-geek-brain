# Инструкции для работы 

## Выделение 

для выделения курсивом необходимо проставить звездочки (*) вначале и в конце слова или предложения. 

Пример: *пам пам*

Для выдления полужирным необходимо поставить двойные звездоччки (**).

Пример:**пам пам**
